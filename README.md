# Кофе-автомат

## Описание
Программа имитирующая поведение кофе-машины. Для взаимодействия с пользователем разработан графический интерфейс (WPF). В качестве базы данных используется SQLite.

## Запуск
Для запуска требуется сборка проекта, дополнительных действий не требуется.

## Возможности
### Главное окно
1) Область выбора позиции.
- Выпадающий список с возможными напитками;
- Кнопка добавления молока к напитку;
- Кнопка добавления сахара к напитку;
2) Область выбранная позиция.
- Текстовое поле отображающее выбранный напиток, кол-во молока, кол-во сахара, цену;
3) Область денег.
- Отображается текущий счет пользователя;
- Отображается сдача(после заказа);
- Выпадающий список где можно выбрать тип платежа (по умолчанию только наличные);
- Выпадающий список с номиналом денежных средств для внесения;
- Кнопка добавляющая деньги на счет пользователя;
4) Область описания.
- Выводит сообщения для ориентации пользователя.
5) Кнопка "Тех панель" - открывает новое окно - "Панель администратора".
6) Кнопка "Загрузить данные" - удаляет все данные о напитках, ресурсах, переводе единиц и заполняет значениями по умолчанию.
7) Кнопка "Заказать" - оформляет выбранный заказ, использует ресурсы, расчитывает сдачу, записыват заказ в базу данных. 
### Панель администратора
1) Область напитков - отображает доступные напитки и их характеристики.
2) Область новый напиток - содержит поля для добавления нового напитка.
- кнопка "Добавить" добавляет новый напиток (обязательно уникальное имя напитка);
- кнопка "Удалить" удаляет напиток выбранный в области "Напитков";
3) Область приведения единиц - позволяет задать значения для перевода 1 единицы ресурса в единицу измерения этого ресурса;
4) Область данные по ресурсам - позволяет добавлять как текущие ресурсы (используемые), так и общие ресурсы кофе-автомата;

ps. Изменения внесенные в панели администратора применяются в главном окне после закрытия первого.
